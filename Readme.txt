Compiling and Running the Application 
•	Open a command prompt/Powershell terminal and navigate to the correct directory (CA folder): cd <directory>
•	Compile the Java files in the terminal:  javac *.java
•	Run a Seller in the terminal: java Seller
•	For each Buyer, open a new terminal and type: java Buyer
•	Buyer input instructions are provided in the terminal window after joining the market
